## Section Component Reference

### Common Components Across Pages

| Component | Used In | Description |
|-----------|---------|-------------|
| Hero | All pages | H1, subheadline, 1-2 CTAs |
| Trust Strip | Home, Hubs | Logos, metrics, badges |
| Card Grid | Hubs, listings | Filterable tile layout |
| Process Steps | Service pages | Numbered step visualization |
| FAQ Accordion | All detail pages | Expandable Q&A |
| CTA Band | All pages | Full-width call-to-action |
| Related Links | Detail pages | Internal linking block |
| Proof Section | Most pages | Case study previews |

---

## Design Consistency Notes

1. **All pages** end with a CTA footer section
2. **All detail pages** include FAQ section for SEO
3. **All hub pages** include filterable grids
4. **Breadcrumbs** appear on all pages except Home
5. **Trust elements** appear early on high-intent pages
